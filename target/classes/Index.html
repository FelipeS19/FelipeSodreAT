Introdução

Esta documentação descreve uma aplicação de loja online desenvolvida para gerenciar clientes, pedidos e produtos. A aplicação foi construída utilizando o framework Spark e a biblioteca Gson em Java para fornecer endpoints RESTful. A seguir, são apresentados os objetivos, as tecnologias utilizadas, uma explicação das classes principais, bem como as funcionalidades implementadas

Objetivo

O objetivo principal da aplicação é fornecer uma plataforma para gerenciamento de clientes, pedidos e produtos de uma loja online. Isso inclui operações como adição, listagem e remoção de clientes, pedidos e produtos

Explicação das Classes:

  • Controllers: Estas classes são responsável por gerenciar as operações,Elas possuem métodos para listar , adicionar ou excluir um objeto existente e reorganizar os IDs após exclusão.
  • Models: A classe responsavel por manter os atributos como ID, nome, telefone, CEP e endereço. Ela também possui métodos para acessar e definir esses atributos.
  • Main.java: Classe principal da aplicação. Configura o servidor Spark e os endpoints para os controladores.

Tecnologias Utilizadas:

  • Java: Linguagem de programação principal utilizada para desenvolver a aplicação.
  • Spark: Framework web utilizado para criar os endpoints RESTful.
  • Gson: Biblioteca Java utilizada para converter objetos Java em JSON e vice-versa.

Funcionalidades Implementadas::

    Gerenciamento
  • Adicionar novos objetos.
  • Listar todos os objetos. .
  • Excluir objetos existentes.
    • Integração com API de Consulta de CEP:
  • Ao adicionar um cliente, a aplicação consulta automaticamente a API do ViaCEP para obter o endereço correspondente ao CEP fornecido.
